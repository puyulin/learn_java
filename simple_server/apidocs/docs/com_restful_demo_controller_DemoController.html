1.1.1- 获取用户信息

method：get
url: /queryUserByName

参数列表

参数名类型必需描述
userNamestringtrue用户名

返回结果: Android Code | IOS Code

{
  "username": "string //用户名",
  "cnname": "string //中文名",
  "sex": "int //性别【0:男，1：女】",
  "userPro": {
    "idcard": "string //身份证",
    "address": "string //家庭地址",
    "age": "int //年龄"
  }
}
